# Dotfiles
Version of dotfiles with a smaller footprint. Only using very popular extensions.

## Dependencies

- `neovim`
    - `lazy.nvim`
    - `Comment.nvim`
    - `gitsigns.nvim`
    - `nvim-web-devicons`
    - `harpoon`
    - `lualine.nvim`
    - `mini.files`
    - `nvim-treesitter`
    - `onedark.nvim`
    - `telescope.nvim`
        - `plenary.nvim`
        - `telescope-fzf-native.nvim`
        - `telescope-ui-select.nvim`
    - `tmux.nvim`
    - `trouble.nvim`
- `tmux`
    - `tpm`
    - `tmux-resurrect`
    - `tmux.nvim`
